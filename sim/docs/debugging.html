Virtual MDS-800 Debugging

Debug Features

Debug features are in the menu "Debug".

Trace (custom)
Trace instructions using custom trigger data.
Trace ON
Start tracing instructions now, until "Trace OFF" or RESET.
Trace OFF
Stop tracing, whatever the trigger.
Dump To Log
Redirect various dumps to the log/stderr instead of pop-ups. Menu item changes to "Dump To GUI".
Dump CPU
Dumps information about the CPU state.
Dump Machine
Dump information about the core machine.
Dump MemSys
Dump information about the memory subsystem.
Dump Core
Dump memory image to a file. Note, this does not contain any ROM images that might be overlayed on memory at the time.
Dump Page
Dump a page (256B) of memory.
Disassemble
Disassemble a segment of memory.
Dump device
Dump information about the specific device. There will be one of these for each add-on device that supports debugging.
